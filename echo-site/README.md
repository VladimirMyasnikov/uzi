# Echo-site — витрина УЗИ

## Как запустить

### Вариант 1: двойной клик (Windows)
Запустите **`start-dev.bat`** в папке `echo-site`. Скрипт установит зависимости и запустит сервер.

### Вариант 2: из терминала
```bash
cd echo-site
npm install
npm run dev
```

Откройте в браузере: **http://localhost:3000**

## Требования
- **Node.js 18** или новее. Проверка: `node -v`
- Скачать: https://nodejs.org/ (рекомендуется LTS)

## Если не запускается
1. **«node не является внутренней командой»** — установите Node.js и перезапустите терминал.
2. **Ошибки при `npm install`** — проверьте интернет; при блокировке npm смените регистр или используйте VPN.
3. **Порт 3000 занят** — запустите на другом порту: `npm run dev -- -p 3001`
4. Пришлите текст ошибки из терминала — по нему можно точнее подсказать.
